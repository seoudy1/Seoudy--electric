SEOUDY ELECTRIC - Professional static site (Arabic)
--------------------------------------------------
Files included:
- index.html, services.html, about.html, contact.html
- styles.css
- assets/ (logo, banner, ad, favicon)
This is a static site ready for upload to Vercel/Netlify/GitHub Pages.

Quick upload to Vercel (recommended on desktop):
1. Go to https://vercel.com and sign in.
2. New Project -> Import -> Upload the project folder (or use GitHub).
3. If static site, no build command needed.
4. Deploy and get your site URL.

If you want, I can guide you step-by-step to upload from your phone.
